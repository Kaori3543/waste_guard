# 浪費警察（Waste Guard）

## 1. アプリ概要
買い物前に「なぜ欲しいのか？」を入力し、
その理由から「必要度」を簡易判定するアプリ。
浪費を防ぎ、計画的な消費をサポートすることを目的とする。

## 2. 企画背景 / 課題
- 衝動買いによる浪費が増えがち
- 特にスマホの普及により購入ハードルが低下
- 買い物前の「ひと呼吸」を促す仕組みが必要

このアプリで理由の言語化を促進し、
自分の判断を客観視できるようにする。

## 3. ターゲットユーザー
- つい無駄遣いをしてしまう学生・社会人
- ECサイトの“誘惑”に弱い人

## 4. 主な機能（最低限）
| 機能名 | 内容 |
|--------|------|
| 理由入力 | 欲しい理由を1行入力 |
| 必要度判定 | キーワードから簡易スコア算出 |
| 結果表示 | 必要度・簡易アドバイス表示 |

※ 判定は仮ロジックで問題なし（モックアップ）

## 5. 画面構成 / 遷移
Home（理由入力画面）
　└ 判定ボタン押下
Result（結果表示画面）


### 画面詳細
| 画面 | 内容 | Widgetイメージ |
|------|------|----------------|
| Home | TextField, Button | 理由入力＋判定ボタン |
| Result | Text, Color表示 | スコア応じてコメント表示 |

## 6. 判定仕様（暫定）
理由テキストに含まれるワードによりスコア変動：

- 高スコアワード  
  - 「必要」「勉強」「健康」「課題」
- 低スコアワード  
  - 「可愛い」「限定」「安い」「気分」

例：
- スコア 0〜3：浪費の可能性高い
- スコア 4〜6：悩む価値あり 
- スコア 7以上：必要！買ってよし
